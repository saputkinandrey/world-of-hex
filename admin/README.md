# Admin UI (Next.js + MUI)

Simple control panel for Seaâ€‘Combat state.

## Run locally

1. Install dependencies:
   ```
   npm install
   ```
2. Start dev server:
   ```
   npm run dev
   ```

The UI expects the backend at `http://localhost:3000/api` by default.
Override via:

```
NEXT_PUBLIC_API_BASE=http://localhost:3000/api
```

## Build & run

```
npm run build
npm run start
```
